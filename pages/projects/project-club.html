🎓 社团管理系统

📋 项目简介

这是一个基于Spring Boot + Vue.js开发的高校社团管理系统，旨在为高校社团提供一个完整的信息化管理平台。系统实现了社团管理、活动发布、成员管理、消息通知等核心功能，提升了社团管理的效率和规范性。

✨ 核心功能

  • 🏢 社团管理：社团信息维护、社团分类管理、社团审核
  • 👥 成员管理：成员信息管理、角色权限分配、入社申请审批
  • 📅 活动管理：活动发布、活动报名、活动签到、活动统计
  • 📢 公告通知：系统公告、社团通知、消息推送
  • 💰 财务管理：社费缴纳记录、财务统计分析
  • 📊 数据统计：社团数据分析、活动数据统计、可视化报表
  • 💬 实时消息：基于WebSocket的实时消息通知

🛠️ 技术栈

Java Spring Boot MyBatis MySQL Vue.js Element UI WebSocket ECharts

🎯 项目亮点

  • 前后端分离架构，提高开发效率和系统可维护性
  • 基于RBAC的权限管理，支持多角色权限控制
  • WebSocket实时消息推送，提升用户体验
  • ECharts数据可视化，直观展示统计数据
  • 响应式设计，支持PC端和移动端访问
